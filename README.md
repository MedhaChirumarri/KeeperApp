# Keeper App – React Notes Application

A clean and intuitive note-taking application inspired by Google Keep. This project demonstrates my ability to build responsive UI components, manage state effectively, and implement user-focused features using React. 
The live URL of this project is https://s7fnyx.csb.app/

---

## Overview

The Keeper App allows users to create and manage notes dynamically. It focuses on simplicity, usability, and clean design while showcasing core front-end development skills.

---

## Features

* Add notes with a title and content
* Delete notes instantly
* Prevent empty notes from being added
* Real-time updates using React state
* Minimal and user-friendly interface
* Filter notes based on text

---

## Tech Stack

* **Frontend:** React (Hooks – useState)
* **Languages:** JavaScript (ES6+), HTML5, CSS3
* **UI Enhancements:** React Icons
